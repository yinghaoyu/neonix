## BIOS

Basic Input Output System

BIOS 在加点自检将主引导扇区读到物理地址为`0x7c00`位置，并跳转到这里执行。

```s
int 0x10 ; BIOS 系统调用
```

## 实模式

8086模式，16位，保护模式

- Real Mode
- Protected Mode

```s
; 0xb8000 文本显示器的内存区域
mov ax, 0xb800
mov ds, ax
mov byte [0], 'H'
```

## 实模式的寻址方式

> 有效地址 = 段地址 << 4 + 偏移地址

EA = 0xb800 << 0x04 + 0 = 0xb8000

EA(Effective Address)

当时寄存器长为16 bit想要访问1 M的内存，需要20 bit，因此需要把段地址 << 4

32 bit - 4 G

## 主引导扇区结构

- 代码: 446B
- 硬盘分区表：64B = 4 * 16B
- 魔数：0xaa55 - 0x55 0xaa（小端存储）

## 主要功能

读取内核加载器并执行
